# TaiwanBank7DigitCode
台灣銀行與分行7碼代碼JSON

蒐集到2023年5月，常見銀行與其分行的銀行代碼

台灣的匯款系統蠻奇特的，一般匯款時只需要3碼銀行代碼 + 帳號即可。

不過臨櫃或是批次匯款時，是需要 3碼銀行代碼 + 4碼的分行代碼。

分行代碼的查詢，並沒有開放API可使用，所以這邊直接整理成JSON，方便需要開發人員的各位使用
